Welcome to Our Site

This is a Real-Time messaging app that is intended to allow a users to create a profile and sign into the profile. Once a user creates a profile the user should be able to create and join chat rooms. When in the chat room a user can send and receive messages from other users and send to other users that are also signed into the platform.

How To Install

This app is run through a web browser. To get the app running, clone the github repository or extract the .zip file. Next you will need to activate a python environment, and utilizing the ‘pip install -r requirements.txt’ command, you can install all of the dependencies for the project. From there, the only command that needs to be run is ‘python manage.py runserver’, which will allow you to navigate to your local IP address in your browser to see the app.
